# 🏦 Besant Bank — DBMS Banking System Project

## 📌 Overview

**Besant Bank** is a database-driven project that simulates a real-world **banking system**.
It demonstrates core **DBMS concepts**, including account management, transactions, auditing, and data integrity.

This project is designed to reflect how modern banking systems manage **customers, accounts, and financial operations securely and efficiently**.

---

## 🚀 Features

* 👤 Customer Management
* 💳 Account Creation & Management
* 💸 Transaction Handling (Deposit / Withdraw / Transfer)
* 📜 Audit Trail for tracking changes
* ⚡ Optimized queries for performance
* 🔒 Data integrity using constraints & relationships

---

## 📂 Database Structure

### 🔹 Core Tables

* `CUSTOMER` → Stores customer details
* `ACCOUNT` → Bank accounts linked to customers
* `TRANSACTION` → Records all financial operations
* `AUDIT_LOG` → Tracks changes for compliance

---

## 🧠 Concepts Implemented

* ✔️ **Normalization (3NF)**
* ✔️ **Primary & Foreign Keys**
* ✔️ **ACID Transactions**
* ✔️ **Triggers (Audit Trail)**
* ✔️ **Indexes for performance**
* ✔️ **SQL Best Practices**
